## Root Dependencies

### Evidence Before Judgment

Supports:

- Career Stage is not title/production/org chart.
- Professional Excellence is not production/charisma/title.
- Development is not information/activity alone.
- Readiness is not confidence/tenure/training completion.
- Manager input does not override facts.
- Latent demand does not justify invented urgency.

Depends on:

- evidence contracts
- confidence labels
- source ownership
- human review when ambiguous

### Human Dignity Boundary

Supports:

- Not ready is not failure.
- Excellence is not human worth.
- Development is not a ranking.
- Purpose is not pressure.
- Manager Intelligence is not surveillance.

Depends on:

- visibility guardrails
- language discipline
- anti-ranking rules
- no public leaderboard defaults

### Forge Core vs Context Boundary

Supports:

- SMNYL Agency 2026 is not Forge Core.
- Local hierarchy belongs in organization profiles or rule packs.
- SMNYL human development truth remains contextual until universal validation.

Depends on:

- rule pack boundary
- organization profile boundary
- universal vs contextual truth classification
