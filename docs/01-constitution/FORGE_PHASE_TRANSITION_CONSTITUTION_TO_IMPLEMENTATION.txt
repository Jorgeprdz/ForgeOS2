7. CONSTITUTIONAL QA GATE

Every future implementation must answer:

- What ADR governs this behavior?
- What input does it consume?
- Who owns that input?
- What output does it produce?
- Is the output allowed?
- Does it create a recommendation?
- Does it require human review?
- Does it preserve unknown?
- Does it handle stale evidence?
- Does it handle conflicting evidence?
- Does it risk pressure, manipulation or enforcement?
- Does it violate Client First?
- Does it violate Authority Boundary?
- Does it invent truth?

If any answer is unclear, implementation must stop.
