OBJETIVO

Crear un motor universal para proyectos de pólizas que convierta valores actuariales por año a MXN nominal utilizando la proyección anual de la moneda correspondiente.

ARCHIVOS CREADOS

1. shared-policy-currency-timeline-engine.js
   Motor compartido para UDI y USD.

2. shared-policy-currency-timeline-smoke-test.js
   Prueba humo con caso Imagina Ser.

3. SHARED_POLICY_CURRENCY_TIMELINE_ENGINE_REPORT.txt
   Reporte de resultados, valores anuales y totales nominales.

ALCANCE

Productos objetivo:
- Imagina Ser
- Orvi
- Vida Mujer
- Nuevo Plenitud
- Segubeca

Este motor queda listo para integrarse con otros motores de Forge.
No se modificó UI.
No se modificó Build Tree.
No se modificaron productos específicos.

REGLAS IMPLEMENTADAS

1. No se usa la UDI o el dólar actual para todos los años.
   Cada año usa su valor de moneda proyectado.

2. Soporta UDI y USD.

3. Soporta escenarios:
   - BASE
   - FAVORABLE
   - DESFAVORABLE

4. Soporta valores anuales y mensualidades.

5. Para mensualidades:
   - conserva valor nominal mensual;
   - calcula valor nominal anualizado.

6. Todos los resultados proyectados usan etiqueta:
   POLICY_CURRENCY_TIMELINE_PROJECTED_MXN_ESTIMATE_NOT_GUARANTEED

7. Disclosure:
   VALORES_NOMINALES_PROYECTADOS_NO_GARANTIZADOS
