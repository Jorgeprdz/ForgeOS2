# Forge GMM Human Review Evidence

Status: PRE-IMPLEMENTATION / FINAL PRE-ENGINE DISCOVERY

## Purpose

Define what evidence resolves each human review gate.

## Preexistence Ambiguity

Gate:

- Condition may have existed before policy.

Evidence needed:

- Prior medical records.
- Prior diagnosis date.
- Prior diagnostic studies.
- Prior treatment records.
- Prior expense history.
- Application declaration.

Resolved by:

- Human medical/policy review.

## Date Conflict

Gate:

- Symptom, diagnosis, accident, first expense or first attention dates conflict.

Evidence needed:

- Clinical file.
- Hospital admission record.
- First consultation record.
- Invoice dates.
- Accident report.

Resolved by:

- Chronology review.

## Foreign Optional Coverage

Gate:

- Foreign care requires optional coverage.

Evidence needed:

- Caratula.
- Endorsement.
- Optional coverage document.
- Foreign care date/location.
- Programmed/urgent status.
- Stay duration proof.

Resolved by:

- Policy/endorsement review.

## High-Specialty Authorization

Gate:

- Treatment requires authorization or second valuation.

Evidence needed:

- Event Programmed approval.
- Prior written authorization.
- Second valuation.
- Specialist report.
- Provider certification.
- COFEPRIS evidence when applicable.

Resolved by:

- Medical/authorization review.

## Source Conflict

Gate:

- Sources disagree.

Evidence needed:

- Caratula.
- Endorsement.
- Active conditions.
- Source versions.
- Effective dates.
- Replacement source if any.

Resolved by:

- Source hierarchy and compliance review.

## Alcohol / Drug / Intentional Conduct

Gate:

- Accident may trigger exclusion.

Evidence needed:

- Accident report.
- Toxicology/alcohol evidence.
- Local legal threshold.
- Prescription record if medication involved.
- Responsibility facts.

Resolved by:

- Human/legal review.

## Professional Sport / Racing / Dangerous Activity

Gate:

- Activity context may affect coverage.

Evidence needed:

- Activity description.
- Competition/racing evidence.
- Remuneration/professional status.
- Accident report.

Resolved by:

- Human review.

## Maternity / Assisted Reproduction Ambiguity

Gate:

- Maternity pathway depends on dates or assisted reproduction status.

Evidence needed:

- Expected birth/birth date.
- Coverage continuity.
- Assisted reproduction method.
- First-trimester notice.
- Institution approval.
- Clinical history.

Resolved by:

- Human/policy review.

## Hospital / Provider Classification

Gate:

- Hospital level or provider status unknown.

Evidence needed:

- Active hospital directory.
- Provider directory.
- Service date.
- Contracted hospital level.

Resolved by:

- Operational catalog review.
