# SG-001 Scaffold Source of Truth Index

## Purpose

Register only conceptual owners already established by the Unified Constitution or active ADR lineage for use by future scaffold documents.

## Scope

Owner-reference index only. It does not create owners, source records, domain implementations or populated domain scaffolds.

## Responsibilities

| Concept | Established conceptual owner | Active authority | Registration limit |
|---|---|---|---|
| Product Truth | Product Intelligence | ADR-005 | Product truth only; not policy, suitability, compensation or recommendation. |
| Policy Truth | Policy Intelligence | ADR-006 | Specific policy state/content/events only. |
| Forecast scenarios | Forecast Intelligence | ADR-007 | Scenarios, not facts. |
| Relationship signals | Relationship Intelligence | ADR-011 | Signals/context only; relationship is not permission. |
| Action paths | Business Planning | ADR-012 | Action paths, not source truth or execution. |
| Observable behavior patterns | Mick | ADR-013 | Observable patterns, not productivity, psychology or human worth. |
| Productivity metrics | Productivity | ADR-014 | Activity/throughput/conversion/output metrics within its boundary. |
| Manager-facing coaching context | Manager Intelligence | ADR-015 | Coaching and attention context; no human consequences. |
| Advisor-facing explanation and feature learning state | Advisor Experience | ADR-016 | Capability-building experience; no dependency or hidden authority. |
| First-value experience | Benvenù | ADR-016 | First value only; not CRM intake or excessive capture. |
| Compensation rule interpretation | Compensation Intelligence | ADR-017 | Rule-based interpretation; no invented or payment truth. |
| Responsible economic motivation presentation | Economic Motivation | ADR-018 | Context only; Client First remains superior. |

Cross-cutting rules:

- ADR-001 governs evidence ownership and source validity; it does not make every evidence holder a truth owner.
- ADR-002 requires one conceptual owner per official metric; it does not pre-assign unknown metric owners.
- ADR-003 preserves accountable human decision authority.
- ADR-004 prohibits invented recommendations.
- ADR-009 states NBA creates neither truth nor authority.
- ADR-010 bounds NASH conversation guidance; it may not invent upstream truth or execute communication.
- ADR-020 provides the architecture baseline without creating runtime or missing domain owners.

## Authority

Unified Constitution Articles III, VI and VII; ADR-001–018 closure-evidenced lineage; ADR-020; SG-001.

## Boundaries

- No owner absent from explicit current authority is registered.
- This index does not resolve ambiguous ownership.
- ADR-016A, ADR-0019, SKYNET and Russell create no owner entry.
- Directory names, code, historical documents and Build Tree status create no owner.

## Dependencies

Governance Registry, Constitution Map, active ADR lineage and the canonical Source of Truth scaffold.

## Source of Truth

The cited Constitution/ADR owns each definition. This index owns navigation to those established owners only.
