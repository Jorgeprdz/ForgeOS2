## Forge Northstar

Forge exists to convert signals into voluntary explained action.

It is not enough to say what to do. Forge must explain:

- why do it
- why now
- why this person
- why this action
- why this message

NBA means Next Best Action plus Reason Why.

- Nash provides conversation, objections, style, message angle, and Nash Combat support.
- Mick provides behavior, execution, consistency, habits, patterns, and why-now / why-this-action context.
- Learning / ML / DL may later learn which messages work better and each advisor's communication style, but only inside the Constitution.

## Psychology Rule

Orders create resistance.

Explanation creates voluntary action.

Forge must not command. Forge must explain the action clearly enough that the human chooses to act. Reason Why must support clarity and confidence, not coercion.

## NBA Reason Why Contract Concept

The future NBA Reason Why contract should consume protected/sanitized context only and produce a non-executing recommendation.

The contract should answer:

- What action could be considered?
- Why could this action matter?
- Why now?
- Why this person?
- Why this message angle?
- What evidence supports the context?
- What is missing, stale, blocked, or unknown?
- What must remain human-reviewed?

The contract must not:

- execute the action
- create a task
- create a calendar event
- send a message
- call Nash legacy runtime
- call Mick runtime
- call an LLM
- approve a draft
- create compensation/revenue/payout/lifecycle/ranking/punishment/HR truth
