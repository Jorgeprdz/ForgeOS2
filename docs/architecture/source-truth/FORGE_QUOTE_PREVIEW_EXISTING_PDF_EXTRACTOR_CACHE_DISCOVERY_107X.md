## En humano

107X is a discovery phase.

Before creating any new PDF bridge, Forge checks the repository for existing PDF extractor and PDF cache modules.

The correct product route is cache-first:

PDF upload or selected PDF

-> existing extractor

-> existing extraction cache

-> confirmation popup reads cache

-> UI reads confirmed cache

The user does not transcribe PDF values.
