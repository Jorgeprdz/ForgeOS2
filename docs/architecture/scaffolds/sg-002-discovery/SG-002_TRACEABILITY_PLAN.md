# SG-002 Traceability Plan

`STATUS=DRAFT`

`AUTHORIZATION=NOT AUTHORIZED`

`RATIFICATION=NOT RATIFIED`

`IMPLEMENTATION_AUTHORITY=NO IMPLEMENTATION AUTHORITY`

## Required Chain Per Instance

```text
Unified Constitution
  -> owner-establishing ADR
  -> ratified SG-002 gate and exact filename manifest
  -> SG-001 canonical scaffold
  -> instantiated document
  -> SG-002 inventory/index
  -> Build Tree evidence-backed status
```

## Required Evidence Fields

- Exact constitutional article.
- Exact ADR decision and owner wording.
- SG-001 template version.
- Historical evidence used, if any, labeled non-normative.
- Differences from historical architecture.
- Upstream owners and downstream consumers.
- Unknowns, exclusions and human authority.
- Version, status and provenance.

No instance may cite code, filenames, directory placement or Build Tree status as owner-creating authority.
