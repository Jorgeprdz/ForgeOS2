# Manager Intelligence Domain Responsibility

## Purpose
Document Manager Intelligence ownership of manager-facing coaching context without human-consequence authority.
## Scope
Evidence-bounded coaching, attention, development and review context for managers.
## Responsibilities
Consume official signals; explain coaching context with provenance, uncertainty, privacy and human review; prevent advisor-facing leakage.
## Authority
Unified Constitution Articles IV and VI; ADR-015; ADR-020; SG-002 Gate. Owner evidence: ADR-015.
## Boundaries
Owns coaching context, not source metrics, hiring, firing, promotion, punishment, ranking, compensation or human potential. Managers retain decisions.
## Dependencies
Official outputs from their domain owners, role/purpose controls and accountable manager review.
