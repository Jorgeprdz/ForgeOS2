# Manager Intelligence Authority Boundary
## Purpose
Keep Manager Intelligence inside coaching support and outside automated human consequences.
## Scope
Manager-facing coaching context, sensitive inputs, role/purpose controls and decisions.
## Responsibilities
Preserve evidence/ownership/privacy; require accountable manager review; block leakage and automated consequences.
## Authority
Unified Constitution Articles IV/VI; ADR-015; ADR-020; SG-002 Gate. Owner evidence: ADR-015.
## Boundaries
Permitted: coaching context. Prohibited: hiring/firing/promotion/punishment/ranking, metric redefinition or advisor-facing leakage. Managers retain authority; failure is `UNKNOWN / BLOCKED`.
## Dependencies
Official signals upstream; accountable manager review downstream; no consumer authority transfer.
