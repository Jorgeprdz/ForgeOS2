# Canon Backlog

Each concept must define:

- Canonical name
- Definition
- Owner
- Producer
- Consumer
- Lifecycle
- Source requirements
- Evidence requirements
- Uncertainty semantics
- Human authority boundary
- Prohibited transitions
- Referenced ADRs
- Testable invariants
