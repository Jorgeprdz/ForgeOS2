# Forge Governance

This directory is the canonical governance entry point for ForgeOS.

Authority order:

1. Constitution
2. Constitution Map
3. Canonical ADRs
4. Governance Registry
5. Build Tree and readiness artifacts
6. Human approvals required by policy

The files under this directory do not silently replace existing canonical
documents. `SOURCE_MAP.md` records their current repository locations until a
separate, reviewed governance migration is approved.
