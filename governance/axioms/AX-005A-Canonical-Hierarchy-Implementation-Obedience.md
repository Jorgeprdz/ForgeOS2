# AX-005A Canonical Hierarchy and Implementation Obedience

Status: `RATIFIED / ACTIVE`

Supersedes: `AX-005-Implementation-Must-Obey.md`

Implementation is subordinate to the canonical hierarchy defined by Article II of `governance/constitution/CONSTITUTION_UNIFIED.md` and to every applicable ratified record within its scope.

ROBOCOP and Miranda validate compliance; they do not create a competing hierarchy. If implementation conflicts with applicable authority, implementation is incorrect and must HOLD until corrected.
